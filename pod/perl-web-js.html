perl-web-js
~~~~~~~~~~~

A (temporary) wrapper for JavaScript support in Perl.

* Module

Web::JS::Checker

  A JavaScript syntax conformance checker

* Dependency

Perl 5.8 or later is required.

In addition, the module requires a backend JavaScript implementation.

At the time of writing, the module only supports JE as the backend.
Future revisions of the module might support other backends.
