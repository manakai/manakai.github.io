NAME

Web::Driver::Client::Response

WebDriver response

SYNOPSIS
  $session->execute (q{
    return document.title;
  })->then (sub {
    my $res = $_[0];
    warn $res->json->{value};
  });

DESCRIPTION

The Web::Driver::Client::Response object represents a response from the WebDriver server.

METHODS

There are following methods:

$value = $res->json

Return the Perl value converted from the JSON data in the response, if any, or undef otherwise.

$boolean = $res->is_error

Return whether the response represents an error or not.

$string = '' . $res
$string = $res->stringify

Return a short string representing the response. This should be useful for development.
