NAME

Web::Temma::Tokenizer - A tokenizer for Temma::Parser

DESCRIPTION

The Web::Temma::Tokenizer is a tokenizer for the Temma::Parser module in the Temma package <https://github.com/wakaba/temma>. It should not be used for any other purpose.

SEE ALSO

Temma::Parser module in the Temma package <https://github.com/wakaba/temma>.
