Web::CSS::MediaQueries::Features

Media feature definitions

DESCRIPTION

The Web::CSS::MediaQueries::Features module contains definitions for the media features used in Media Queries. It is used by Web::CSS::MediaQueries::Parser.

SPECIFICATIONS

Media Queries <http://w3c-test.org/csswg/mediaqueries/#media1>.

The 'view-mode' Media Feature <http://dev.w3.org/2006/waf/widgets-vmmf/#the--view-mode--media-feature>.
