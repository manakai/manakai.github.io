Web::CSS

CSS, Selectors, and Media Queries

MODULES

Web::CSS::Parser, Web::CSS::Serializer - CSS syntax

Web::CSS::Values, Web::CSS::Colors, Web::CSS::Props - CSS definitions and microsyntax implementations

Web::CSS::Selectors::Parser, Web::CSS::Selectors::Serializer - Selectors syntax

Web::CSS::Selectors::API - Selectors API implementation

Web::CSS::MediaQueries::Parser, Web::CSS::MediaQueries::Serializer - Media Queries syntax

Web::CSS::MediaQueries::Features - Media Queries feature definitions

Web::CSS::Context, Web::CSS::MediaResolver - Support modules

Unless you are trying to develop a Web browser, you should not use these modules directly. The perl-web-dom package <https://github.com/manakai/perl-web-dom> contains an implementation of the standard CSSOM API and Selectors API using these modules.

DEPENDENCY

These modules require Perl 5.8 or later.

The parse_byte_string method in the Web::CSS::Parser module requires the perl-web-encodings package <https://github.com/manakai/perl-web-encodings>.

SPECIFICATIONS

For the list of supported features and relevant specifications, see the documentation of the modules. In general, they support latest CSS, Selectors, and/or Media Queries standards.
