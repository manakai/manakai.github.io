Web::DOM::Parser

The DOM DOMParser object

DESCRIPTION

The Web::DOM::Parser class implements the DOM DOMParser interface.

METHODS

The class has following methods:

$parser = new Web::DOM::Parser

Create a new DOMParser object. [DOMPARSING]

$doc = $parser->parse_from_string ($string, $type)

Parse the given character string as an HTML or XML document and return the Web::DOM::Document object for the document. The second argument must specify the MIME type of the document, which is one of followings: text/html, text/xml, application/xml, application/xhtml+xml, or images/svg+xml. If the document is an XML document and is not well-formed, a short document containing the error message is returned instead. [DOMPARSING]

See also "DEPENDENCY" in Web::DOM.

SPECIFICATION

DOMPARSING

DOM Parsing and Serialization Standard <http://domparsing.spec.whatwg.org/>.
