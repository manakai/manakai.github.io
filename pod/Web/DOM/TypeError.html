Web::DOM::TypeError

The WebIDL TypeError object

DESCRIPTION

The Web::DOM::TypeError class implements the WebIDL TypeError object.

METHODS

The Web::DOM::TypeError class inherits from Web::DOM::Error. [DOMPERL]

In addition, the Web::DOM::TypeError constructor can be used to create a new instance of the Web::DOM::TypeError object. If the argument is specified, it is used as the message for the object. [DOMPERL]

SPECIFICATIONS

WEBIDL

WebIDL <http://dev.w3.org/2006/webapi/WebIDL/#dfn-predefined-exception>.

ECMASCRIPT
