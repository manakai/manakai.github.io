Web::DOM::AtomIconElement

The DOM AtomIconElement object

DESCRIPTION

The Web::DOM::AtomIconElement class implements the DOM AtomIconElement interface. The class is used by the Atom icon element.

METHODS

The methods for classes Web::DOM::Element and Web::DOM::AtomElement are available.

SPECIFICATION

ATOMDOM

Atom DOM <http://suika.suikawiki.org/www/markup/xml/atomdom>.
