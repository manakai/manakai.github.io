Web::DOM::Error

The base class for errors

DESCRIPTION

The Web::DOM::Error object represents an error or exception in the Web platform.

METHODS

Following methods are available:

$string = $error->name

Return the name of the error, e.g. TimeoutError. [DOMPERL]

$string = $error->message

Return a message describing the error. This message can be used for debugging purpose. The value returned by this method is implementation-dependent. Applications should not rely on the value returned by this method. [DOMPERL]

$string = $error->file_name

Return the file name of the location of the error. [DOMPERL]

$number = $error->line_number

Return the line number, in the file, of the location of the error. [DOMPERL]

$string = '' . $error

Stringifying the error object returns the error message with its location in the Perl standard format (i.e. Error message at path/to/script.pl line 123.\n). [DOMPERL]
