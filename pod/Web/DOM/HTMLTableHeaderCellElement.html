Web::DOM::HTMLTableHeaderCellElement

The DOM HTMLTableHeaderCellElement object

DESCRIPTION

The Web::DOM::HTMLTableHeaderCellElement class implements the DOM HTMLTableHeaderCellElement interface. The class is used by the HTML th element.

METHODS

In addition to the methods available for classes Web::DOM::Element, Web::DOM::HTMLElement, and Web::DOM::HTMLTableCellElement, the following method is available:

$string = $el->abbr
$el->abbr ($string)

Return or set the abbr attribute. [HTML]

$string = $el->scope
$el->scope ($string)

Return or set the scope attribute. [HTML]

$string = $el->sorted
$el->sorted ($string)

Return or set the sorted attribute. [HTML]

SPECIFICATION

HTML

HTML Standard <http://www.whatwg.org/specs/web-apps/current-work/#the-th-element>.
