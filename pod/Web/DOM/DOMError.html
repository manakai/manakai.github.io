Web::DOM::DOMError

The DOM DOMError object

DESCRIPTION

The Web::DOM::DOMError class implements the DOM DOMError interface.

METHODS

The Web::DOM::DOMError class defines following method:

$error = new Web::DOM::DOMError $name, $message

Create a new DOM error object with the specified name. [DOM]

$string = $error->message

Return the implementation-dependent error message. [DOM]

$string = $error->name

Return the name of the error. [DOM]

SPECIFICATION

DOM

DOM Living Standard <http://dom.spec.whatwg.org/#interface-domerror>.

SEE ALSO

Classes Web::DOM::Error and Web::DOM::Exception are irrelevant to this class. Also, DOM3's DOMError interface is different from the current DOM Standard's DOMError interface, which is implemented by this class.
