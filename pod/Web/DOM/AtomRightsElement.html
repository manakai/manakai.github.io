Web::DOM::AtomRightsElement

The DOM AtomRightsElement object

DESCRIPTION

The Web::DOM::AtomRightsElement class implements the DOM AtomRightsElement interface. The class is used by the Atom rights element.

METHODS

The methods for classes Web::DOM::Element, Web::DOM::AtomElement, and Web::DOM::AtomTextConstruct are available.

SPECIFICATION

ATOMDOM

Atom DOM <http://suika.suikawiki.org/www/markup/xml/atomdom>.

SEE ALSO

Web::DOM::AtomElement, Web::DOM::AtomTextConstruct.
