Web::DOM::HTMLInputElement

The DOM HTMLInputElement object

DESCRIPTION

The Web::DOM::HTMLInputElement class implements the DOM HTMLInputElement interface. The class is used by the HTML input element.

METHODS

In addition to the methods available for classes Web::DOM::Element and Web::DOM::HTMLElement, the following methods are available:
