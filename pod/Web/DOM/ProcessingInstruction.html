Web::DOM::ProcessingInstruction

The DOM ProcessingInstruction object

DESCRIPTION

The Web::DOM::ProcessingInstruction class implements the DOM ProcessingInstruction interface. It represents a processing instruction.

METHODS

In addition to the methods of classes Web::DOM::EventTarget, Web::DOM::Node, Web::DOM::CharacterData, and Web::DOM::ChildNode, the following method is available:

$name = $pi->target

Return the target of the processing instruction.

In addition, the obsolete manakai_base_uri method is available. [DOMDTDEF]
