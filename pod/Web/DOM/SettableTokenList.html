Web::DOM::SettableTokenList

The DOM DOMSettableTokenList object

DESCRIPTION

The Web::DOM::SettableTokenList class implements the DOM DOMSettableTokenList interface.

The DOMSettableTokenList object is used by, e.g., HTML sandbox, dropzone, and headers attributes.

METHODS

In addition to the methods available for the class Web::DOM::TokenList, the followinig method is available:

$string = $tokens->value

Return the serialized representation of the tokens, joined by the space character. [DOM]

$tokens->value ($string)

Replace the tokens in the list by the tokens specified as the space-separeted list. [DOM]

SPECIFICATION

DOM

DOM Living Standard <http://dom.spec.whatwg.org/>.
