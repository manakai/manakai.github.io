Web::DOM::HTMLPictureElement

DOM HTMLPictureElement object

DESCRIPTION

The Web::DOM::HTMLPictureElement class implements the DOM HTMLPictureElement interface. This class is used by HTML picture elements.

METHODS

The methods available for classes Web::DOM::Element and Web::DOM::HTMLElement are avaiable.

SPECIFICATION

HTML

HTML Standard <http://www.whatwg.org/specs/web-apps/current-work/>.
