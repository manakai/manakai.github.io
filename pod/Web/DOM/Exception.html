Web::DOM::Exception

The DOM DOMException object

DESCRIPTION

The Web::DOM::Exception class implements the DOM DOMException interface.

METHODS

The Web::DOM::Exception class inherits from Web::DOM::Error. In addition, it defines following method:

$code = $error->code

Return the error code. This method is considered depreacated and the name method should be used instead. [DOM]

CONSTANTS

This class exports several constants for error codes, but they are now considered useless in favor of name method taking string error names.

SPECIFICATION

DOM

DOM Living Standard - Exception DOMException <http://dom.spec.whatwg.org/#exception-domexception>.
