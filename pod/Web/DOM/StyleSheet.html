Web::DOM::StyleSheet

The DOM StyleSheet object

DESCRIPTION

The Web::DOM::StyleSheet class implements the DOM StyleSheet interface. It represents a style sheet.

The Web::DOM::StyleSheet class is not used standalone. See the subclass, Web::DOM::CSSStyleSheet.

SPECIFICATION

CSSOM

CSS Object Model (CSSOM) <http://dev.w3.org/csswg/cssom/>.

SEE ALSO

Web::DOM::CSSStyleSheet.
