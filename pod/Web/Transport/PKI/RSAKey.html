Web::Transport::PKI::RSAKey

An RSA key

DESCRIPTION

The Web::Transport::PKI::RSAKey module represents an RSA key.

METHOD

There is following method:

$string = $rsa->to_pem

Return the .pem serialization of the RSA private key.
