Web::IPAddr::Canonicalize

Canonicalize IPv4 and IPv6 addresses

SYNOPSIS

  use Web::IPAddr::Canonicalize;
  print canonicalize_ipv4_addr $ipv4_addr;
  print canonicalize_ipv6_addr $ipv6_addr;

DESCRIPTION

The Web::IPAddr::Canonicalize module exports two functions to canonicalize IP addresses as Web browsers do.

FUNCTIONS

This module exports two functions:

$result = canonicalize_ipv4_addr $input

Canonicalize the input string as an IPv4 address. If the input is not an IPv4 address, undef is returned.

$result = canonicalize_ipv6_addr $input

Canonicalize the input string as an IPv6 address. If the input is not an IPv6 address, undef is returned.
