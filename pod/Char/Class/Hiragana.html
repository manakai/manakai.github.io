NAME

Char::Class::Hiragana - Regular Expression Character Classes - Hiragana

COLLECTION NAMES

  • InHiraganaAll
  • InHiraganaDigraph
  • InHiraganaHatsuon
  • InHiraganaSeion
  • InHiraganaSemiVoiced
  • InHiraganaSmall

EXAMPLE

 if ($s =~ /\p{InHiraganaAll}/) {
   print "Match!\n";
 }
