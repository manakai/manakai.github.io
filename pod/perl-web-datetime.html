Web::DateTime

Date and time for Web

SYNOPSIS

  use Web::DateTime;
  $dt = Web::DateTime->new->parse_global_date_and_time_string ($value);
  warn $dt->to_unix_integer;

DESCRIPTION

The Web::DateTime module has parsers and serializers of various date and time representations used in Web.
